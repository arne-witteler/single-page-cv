# Single-Page CV

This project is a simple, fully semantic **single-page CV** built using **only HTML**.  
It includes SEO meta tags, Open Graph (OG) tags for social sharing, and a favicon.  
The goal is to provide a clean HTML foundation that can later be extended with CSS.
